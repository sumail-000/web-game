# Background Images

## Adding a Custom Background

To add a custom game background image for the signup and login pages:

1. Place your background image in this directory (`frontend/public/images/`)
2. Name it `game-background.jpg` (or update the path in the code)
3. The image will automatically be used with a blur and brightness filter

## Recommended Image Specifications

- **Format:** JPG or PNG
- **Resolution:** 1920x1080 or higher
- **Aspect Ratio:** 16:9
- **File Size:** Keep under 500KB for faster loading
- **Content:** Colorful game scenes work best

## Current Setup

Currently, the pages use an animated gradient background as a placeholder. Once you add `game-background.jpg`, it will automatically replace the gradient.

## Alternative: Using the Gradient

If you prefer to keep the animated gradient background, simply don't add a `game-background.jpg` file. The gradient provides a nice Roblox-inspired colorful background effect.
